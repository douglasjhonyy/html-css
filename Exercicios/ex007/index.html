Formatos de Textos

existe 2 maneiras de colocar seu texto em Negrito


1. usar a tag <i>
2. usar a tag <strong>
ambas funcionam porem a 1 não é semântica, já 2 sim

Tambem existe 2 maneiras de colocar o texto em Itálico
1. usar a tag <i>
2. usar a tag <em>

Atalhos que vão ajudar

caso você esqueça de colocar alguma tag, não precisa voltar e fazer tudo novamente, basta selecionar a palavra de digitar o comando

CRTL + SHIFT + P = escolher a opção wrap with abbreviation


caso não ache a opção basta digitar ''abb''

você tambem pode utilizar o atalho ainda mais fácil, usando o comando


CRTL + SHIFT + P + ENTER + ''o comando desejado'' (exemple: p, b , i )

Texto Marcado

podemos criar um texto marcado usando a tag <mark> caso você queira mudar a cor, deve abri uma tag style

<mark style=backgroundcolor= ''cor escolhida''
