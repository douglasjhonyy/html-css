Outras Formatações

Códigos Fontes

quando o texto for em formato de um código, você pode usar a tag code
<code>

quando for um código com identação, usar a tag pre
<pre>
Exemplo 

    mytuple = ("apple", "banana", "cherry")
    myit = iter(mytuple)
    
    print(next(myit))
    print(next(myit))
    print(next(myit))

caso estiver com muito espaço, você seleciona o texto e pode usar o atalho (SHIFT+TAB)
mas se quiser um espaço ou uma identação, basta selecionar o texto e clicar em TAB

Citações Simple

Em uma citação você pode simplesmente usar as aspas, ou usar tag <q>

usar apenas quando for uma citação

Citação Completa

usando a tag ''blockquote'' permite que a citação apareça deslocada

Ex:

Tudo o que um sonho precisa para ser realizado é alguém que acredite que ele possa ser realizado
.

você tambem pode criar um paramentro ''cite'' e adicionar o link da citação
Ex: blockquote cite=''link da citação''
