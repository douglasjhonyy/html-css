Usando Links externos

Para você colocar uma tag na palavra, basta selecionar e digitar o comando
<a href=''LINK'' >

Mas para evitar que o link faça sair na sua página, basta criar um parametro
''target=''_blank'' rel=''external''
Ex:
<a> href=''link'' target=''_blank'' rel=''external''
