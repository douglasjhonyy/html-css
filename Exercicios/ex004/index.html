Vou coloca um favion no site


-salvar uma imagem em formato icon


Você deve clicar na linha acima da tag ''TITLE'' e abrir um tag ''LINK:FAVICON''


Depois você deve renomear a opção ''href=favicon.ico'' clicando com o comando ''CRTL+ESPAÇO'' e indicar o arquivo


Aqui estõa alguns site para gerar,criar ou converter ICON


- Iconarchive = para salvar em formato .ico
- Favicon cc = para criar
- Favicon.io = para gerar ou converter
